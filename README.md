# MeetX — Connect Beyond Limits

A modern full-stack real-time video conferencing and collaboration platform built using the MERN stack. MeetX enables users to securely create and join meetings with real-time communication features, responsive UI, and scalable backend architecture.

---

## 🚀 Features

- 🔐 Secure Authentication using JWT and bcryptjs
- 🎥 Real-Time Video & Audio Communication
- 💬 Real-Time Chat Functionality
- 👥 Create & Join Meeting Rooms
- 🌐 Responsive Modern UI/UX
- ⚡ RESTful API Architecture
- ☁️ MongoDB Atlas Cloud Database Integration
- 🚀 Frontend & Backend Deployment Support
- 📱 Cross-Device Responsive Design

---

## 🛠️ Tech Stack

### Frontend
